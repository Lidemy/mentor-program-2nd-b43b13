## 請列舉出 5 個 command line 指令並且說明功用

- pwd (print working directory) 看現在位置
- ls (list) 列出檔案清單
- cd (change directory) 切換現在資料夾
- rm (remove) 刪除檔案
- man(manual) => 看使用手冊
